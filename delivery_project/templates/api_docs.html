Orders API

The Orders API allows you to create, manage, and track deliveries programmatically via REST.

GET

List all orders

Returns a paginated list of all active and historical orders associated with your account.

Endpoint

https://api.globaldash.com/v1/orders

Example Response

  "object": "list",
